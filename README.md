# How to use GigaLearnCPP (RLBot Tutorial)

A simple guide that teaches you how to make Rocket League bots using GigaLearnCPP.
This guide is written for beginners. No understanding of math needed. No coding skill required. All of the following is simplified for beginners.

## WHAT THIS GUIDE GIVES YOU:

* A full walkthrough of the GigaLearn repo
* How training a bot works
* How rewards work + weighing rewards
* How observations (OBS) work
* How to understand PPO in simple terms
* How to avoid impractical mistakes
* An insight on how your bot develops overtime

## WHO THIS GUIDE IS FOR:

* Anyone new to GigaLearn/PPO
* Anyone confused by the original codebase
* Anyone who wants a clean step by step path
* P.S: This does NOT cover the installation of GigaLearn; this is for people who already have it working

## HOW TO USE THIS GUIDE:

1. Start from the beginning.
2. Read each section in order.
3. Try the examples and information given.
4. Follow the short steps.
5. Do not skip ahead unless you understand the basics.

## SECTIONS

* 01: Overview
* 02: Observations
* 03: Rewards
* 04: ExampleMain.cpp
* 05: StateSetters
* 06: Training

Each folder will have simple markdown files with short explanations; nothing more.
